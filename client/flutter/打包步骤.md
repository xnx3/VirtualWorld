# Genesis Flutter App 本地打包步骤

## 一、环境准备

### 1. 安装 Flutter SDK

**Windows:**
1. 下载: https://docs.flutter.dev/get-started/install/windows
2. 解压到 `C:\flutter`
3. 添加到 PATH: `C:\flutter\bin`

**macOS:**
```bash
# 使用 Homebrew
brew install flutter

# 或手动下载
# 下载地址: https://docs.flutter.dev/get-started/install/macos
```

**Linux:**
```bash
# 下载并解压
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:`pwd`/flutter/bin"
```

### 2. 安装 Android Studio（用于 Android 打包）

1. 下载: https://developer.android.com/studio
2. 安装后打开，进入 SDK Manager
3. 安装 Android SDK 34（或最新版本）
4. 设置环境变量:
   - Windows: `ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk`
   - macOS/Linux: `export ANDROID_HOME=$HOME/Android/Sdk`

### 3. 验证环境

```bash
flutter doctor
```

确保输出中 Android toolchain 显示 ✓

---

## 二、构建 APK

### 1. 复制项目到本地

将服务器上的 `client/flutter` 目录完整复制到本地电脑。

### 2. 进入项目目录

```bash
cd client/flutter
```

### 3. 安装依赖

```bash
flutter pub get
```

### 4. 构建 APK

```bash
flutter build apk --release
```

或使用提供的脚本:
```bash
./build_apk.sh
```

### 5. 获取 APK

构建成功后，APK 位于:
```
build/app/outputs/flutter-apk/app-release.apk
```

---

## 三、安装到手机

### 方式一: ADB 安装

```bash
# 连接手机（开启 USB 调试）
adb install build/app/outputs/flutter-apk/app-release.apk
```

### 方式二: 直接安装

1. 将 APK 文件传输到手机
2. 在手机上点击安装（需允许未知来源）

---

## 四、连接后端

### 1. 启动 Python 后端

在服务器上执行:
```bash
cd /home/git/VirtualWorld
./genesis.sh start --api
```

后端将在 `ws://127.0.0.1:19842` 监听。

### 2. 手机连接后端

**情况A: 手机和电脑同一 WiFi**

1. 查看服务器 IP:
   ```bash
   # Linux/macOS
   ip addr show
   # 或
   ifconfig

   # Windows
   ipconfig
   ```

2. 在 App 设置中修改服务器地址为服务器 IP

3. 确保服务器防火墙开放 19842 端口:
   ```bash
   # Linux
   sudo ufw allow 19842/tcp
   # 或
   sudo firewall-cmd --add-port=19842/tcp --permanent
   sudo firewall-cmd --reload
   ```

**情况B: 本地运行**

如果后端在本地电脑运行，App 默认连接 `127.0.0.1:19842` 即可。

## 五、常见问题

### Q1: flutter doctor 显示 Android licenses 未接受
```bash
flutter doctor --android-licenses
# 然后按 y 接受所有许可
```

### Q2: 构建失败，提示 Gradle 下载超时
修改 `android/build.gradle`，添加国内镜像:
```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        mavenCentral()
    }
}
```

### Q3: APK 体积太大
使用分包构建:
```bash
flutter build apk --split-per-abi
```
会生成 3 个小 APK，分别对应不同 CPU 架构。

### Q4: 手机无法连接后端
1. 确认手机和服务器在同一网络
2. 检查服务器防火墙
3. 在 App 设置页面测试连接

## 六、快速命令汇总

```bash
# 检查环境
flutter doctor

# 安装依赖
flutter pub get

# 构建调试版
flutter build apk --debug

# 构建发布版
flutter build apk --release

# 分架构构建（体积更小）
flutter build apk --split-per-abi

# 安装到手机
adb install build/app/outputs/flutter-apk/app-release.apk

# 清理构建缓存
flutter clean
```

## 七、项目文件清单

构建 APK 需要以下文件:

```
client/flutter/
├── android/              # Android 配置（必须）
├── lib/                  # Dart 源码（必须）
├── assets/               # 静态资源
├── pubspec.yaml          # 项目配置（必须）
├── pubspec.lock          # 依赖锁定
├── analysis_options.yaml # 代码分析
└── build_apk.sh          # 构建脚本
```

确保这些文件完整复制到本地。
